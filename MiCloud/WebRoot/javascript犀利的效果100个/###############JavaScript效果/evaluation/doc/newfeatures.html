New - pinNavigator 2.0

New Features

Find below a list of new or enhanced features. 

Features Description
New Extended browser support

pinNavigator supports now

IE 5.5+,
Firefox 1.0+,
Mozilla 1.7+,
Netscape 7.1+,
Opera 7.2+

Older versions or other browsers might also work but have not been tested right now.
New New style A new tab style has been added with different colors
New ASP.Net feature support pinNavigator supports now initiating of a postback in tab item page and to use client validation functions (Validators)
New Enable/Disable of tab items
New Visible/Unvisible of tab items
New Loading behaviour Pages can be loaded on startup or on demand
New API extended

A lot of new client API functions have been added like
getItemByIndex
getItems
getDocument
getWindow
call
activate
etc.

New Events extended pinNavigator  now raises client events after an item page has been loaded and a event before the "click" event.
Events can be interrupted.
New Rendering optimization Rendering of items has been optimized
New New samples
